Valores y Atributos

En HTML las etiquetas se pueden escribir en mayúscula o minúscula, indistintamente.

Elemento

El bloque contenido entre una etiqueta de inicio y otra de cierre (o una etiqueta sin emparejar) lo llamaremos elemento.
Veremos que lo normal es que unos elementos estén contenidos dentro de otros elementos. De esta forma daremos estructura
al documento.
Un elemento puede tener una etiqueta con atributos.

Atributos

Un atributo especifica algunas características singulares de un elemento o informacion adicional.
